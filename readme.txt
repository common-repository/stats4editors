== Description ==

This Plugin displays simple site stats in the admin toolbar as opposed to from the main admin menu, this allows all users (all who see the toolbar) to view very simple stats.

the plugin displays the number of page views and number of unique visitors (as determined by the visitors IP address) clicking on the toolbar provides a break down of page views by browser type
